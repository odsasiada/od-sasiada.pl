Tonal pill for status and stock. Use `StatusBadge` for order statuses (maps to the Polish labels + tone automatically).

```jsx
<Badge tone="warning" size="sm">Zostało 3</Badge>
<Badge tone="accent">Sezonowe</Badge>
<StatusBadge status="out_for_delivery" />
```

Tones: `neutral` · `brand` · `success` · `warning` · `error` · `accent`. Add `dot` for a status indicator.
