Inline message block for checkout and account flows — soft tonal fills, never an aggressive alarm.

```jsx
<Alert tone="success" title="Dziękujemy! Zamówienie #2024-118 przyjęte.">
  Zadzwonimy, żeby potwierdzić dostawę. Płatność gotówką przy odbiorze.
</Alert>
<Alert tone="error">Minimalna wartość zamówienia to 30,00 zł.</Alert>
```

Tones: `success` · `warning` · `error` · `info`. Set `icon={false}` to hide the glyph.
